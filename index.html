Unity Libs Repository

This page hosts certain assets obtained from different Unity Editor versions. Such assests are primarily useful for Unity modding.

At the moment, the following assets are included:

  • Unity Engine managed DLLs
  • Unity Player Android binaries
  • NuGet packages of Unity Engine DLLs (see UnityEngine.Modules package)

Asset URLS

Both HTTP and HTTPS endpoints are supported.

Unity Engine managed DLLs

https://unity.bepinex.dev/libraries/<unity version>.zip

Replace <unity version> with the Unity version you need.

Unity Player Android binaries

https://unity.bepinex.dev/android/<unity version>.zip

Replace <unity version> with the Unity version you need.

NuGet packages of Unity Engine DLLs

See UnityEngine.Modules package
